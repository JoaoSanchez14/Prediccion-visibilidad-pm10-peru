# Prediccion-visibilidad-pm10-peru
Modelado predictivo de la visibilidad atmosférica (km) a partir de concentraciones de PM10 en tres estaciones de monitoreo de Piura, Perú. Incluye limpieza y tratamiento de outliers, comparación de 5 modelos de regresión (Lineal, KNN, Random Forest, Gradient Boosting, SVR) con validación cruzada y optimización de hiperparámetros con GridSearchCV.
